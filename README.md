# Habit & Task Tracker API

A REST API built with Django Rest Framework and JWT authentication.

## Features
- User registration & login (JWT)
- Habit management
- Habit logs (daily/weekly)
- Task management
- Analytics
- Secure ownership-based access
- PostgreSQL database

## Tech Stack
- Django
- Django REST Framework
- SimpleJWT
- PostgreSQL

## Setup Instructions

1. Clone the repo
2. Create a virtual environment
3. Install dependencies
4. Create `.env` file
5. Run migrations
6. Start server

## API Endpoints
- /api/auth/
- /api/habits/
- /api/tasks/
